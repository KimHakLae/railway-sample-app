# Railway 요리 비서 (Cooking Assistant) - v2.3.1 [UPDATED]

단순한 재고 관리를 넘어, **시스템 전체가 공유하는 전문 식재료 DB**를 기반으로 최적의 레시피를 추천하고 스마트하게 재고를 소모하는 **지능형 주방 관리 솔루션**입니다.

## 🚀 주요 기능

### 1. 스마트 대시보드 (Smart Dashboard)
- **긴급 재고 알림**: 유통기한이 지났거나 3일 이내로 임박한 식재료를 최상단에 노출하여 낭비를 방지합니다.
- **재고 요약**: 현재 보유 중인 전체 식재료 수량과 상태를 직관적인 카드로 제공합니다.
- **애니메이션 효과**: `framer-motion`을 활용한 부드러운 UI 인터랙션으로 프리미엄 경험을 제공합니다.

### 2. 공유 재고 기반 추천 시스템 (Global & Shared Stock) [UPGRADED]
- **통합 재고 분석**: 특정 사용자 개인이 아닌, **전체 사용자가 등록한 모든 재고**를 기반으로 요리를 추천합니다.
- **유효성 검증 (30% 임계값)**: 최소 하나 이상의 재료가 아닌, **필요 재료의 30% 이상**을 보유한 실효성 있는 레시피만 엄선합니다.
- **정예 추천 (TOP 5)**: 너무 많은 선택지 대신, 매칭률이 가장 높은 **상위 5개의 레시피**를 집중적으로 제안합니다.
- **스마트 재고 소모**: '요리하기' 기능을 통해 레시피에 사용된 재료를 자동으로 차감하며, 유통기한이 가장 짧은 재고부터 소모(FIFO)합니다.

### 3. 사용자 및 관리자 모드 (Roles & Access) [STABILIZED]
- **인증 시스템 정합성**: 백엔드 인증 미들웨어와 DB 필드명(`user_id`)을 통일하여 로그인 및 권한 관리의 안정성을 확보했습니다.
- **관리자 권한 확장**: 관리자도 시스템 관리뿐만 아니라 일반 사용자의 요리 도구와 대시보드를 동일하게 사용할 수 있습니다.
- **레이아웃 최적화**: 모바일 하단 네비게이션과 푸터 간의 겹침 현상을 해결하여 버전 정보 가시성을 높였습니다.

### 4. 재고 및 아이템 관리
- **표준 식재료 DB**: 카테고리와 보관 방식이 정의된 아이템 사전을 기반으로 재고를 등록합니다.
- **상세 필터링**: 냉장/냉동 구분, 카테고리별 필터링 및 이름순/유통기한순 등 다양한 정렬 기능을 지원합니다.

## 🛠️ 기술 스택

### Frontend
- **React 19** / **Vite**
- **Tailwind CSS 4** / **Vanilla CSS**
- **Framer Motion** (사용자 경험 강화 애니메이션)
- **Lucide React** / **Heroicons** (아이콘 시스템)

### Backend
- **Node.js** / **Express.js**
- **Prisma ORM** (PostgreSQL 인터페이스)
- **JWT Auth** (보안 인증)
