# Railway 요리 비서 (Cooking Assistant) - v2.0.0

단순한 재고 관리를 넘어, 보유 식재료를 기반으로 최적의 레시피를 추천하고 스마트하게 재고를 소모하는 **지능형 주방 관리 솔루션**입니다.

## 🚀 주요 기능

### 1. 스마트 대시보드 (Smart Dashboard) [NEW]
- **긴급 재고 알림**: 유통기한이 지났거나 3일 이내로 임박한 식재료를 최상단에 노출하여 낭비를 방지합니다.
- **재고 요약**: 현재 보유 중인 전체 식재료 수량과 상태를 직관적인 카드로 제공합니다.
- **애니메이션 효과**: `framer-motion`을 활용한 부드러운 UI 인터랙션으로 프리미엄 경험을 제공합니다.

### 2. 레시피 추천 및 요리 시스템 (Recipe & Cooking) [NEW]
- **보유 식재료 분석**: 현재 냉장고에 있는 재료를 분석하여 바로 만들 수 있는 요리를 추천합니다.
- **유통기한 기반 점수**: 유통기한이 임박한 재료를 사용하는 레시피에 가산점을 부여하여 우선 추천합니다.
- **스마트 재고 소모**: '요리하기' 기능을 통해 레시피에 사용된 재료를 자동으로 차감하며, 유통기한이 가장 짧은 재고부터 선입선출(FIFO) 방식으로 소모합니다.

### 3. 사용자 및 관리자 모드 (Roles & Access)
- **관리자 권한 확장**: 관리자도 시스템 관리(유저 승인, 공지 작성)뿐만 아니라 일반 사용자의 요리 도구와 대시보드를 동일하게 사용할 수 있습니다.
- **하이브리드 내비게이션**: 데스크탑 상단 바와 모바일 하단 바를 통해 관리 콘솔과 요리 도구 사이를 원활하게 전환할 수 있습니다.

### 4. 재고 및 아이템 관리
- **표준 식재료 DB**: 카테고리와 보관 방식이 정의된 아이템 사전을 기반으로 재고를 등록합니다.
- **상세 필터링**: 냉장/냉동 구분, 카테고리별 필터링 및 이름순/유통기한순 등 다양한 정렬 기능을 지원합니다.

## 🛠️ 기술 스택

### Frontend
- **React 19** / **Vite**
- **Tailwind CSS 4** / **Vanilla CSS**
- **Framer Motion** (사용자 경험 강화 애니메이션)
- **Lucide React** / **Heroicons** (아이콘 시스템)

### Backend
- **Node.js** / **Express.js**
- **Prisma ORM** (PostgreSQL 인터페이스)
- **JWT Auth** (보안 인증)
